# 🚀 ShadowMeal: Underground Redistribution Network  

ShadowMeal is a single‑page web app (SPA) built with **HTML, CSS, JavaScript, and React**.  
It simulates a secret rebel dashboard that intercepts surplus food from luxury towers and redistributes it to hunger zones in a mega‑city.  
The design follows a **rebel cyberpunk theme**: dark, gritty, neon‑lit, and hacker‑style.

---

## 🔹 Features  

### 🗺️ Smuggler’s Map  
- Dark-themed city map UI  
- Highlights hunger zones with glowing effects  
- Animated neon routes for drone delivery  
- Simulated drone movement  

### 🍱 Surplus Stash  
- Food listings in grid/card layout  
- Displays food type, source, and countdown timer  
- "Intercept" button to assign drones  

### 🤖 Drone & Volunteer Dashboard  
- Drone status (battery, cargo, condition)  
- Volunteer availability tracking  
- Task assignment system (UI-based)  

### 📊 Impact Stats  
- Meals saved, people fed, food rescued  
- Animated counters  
- Live ticker simulation  

### 🚨 Panic Mode (Boss Key)  
- Instantly switches UI to a fake safe screen  
- Designed for stealth interaction  

---

## 🎨 Design  

- Cyberpunk / hacker-themed UI  
- Dark background with neon colors (purple, green, orange)  
- Glowing effects using CSS  
- Monospace fonts for terminal-style feel  
- Fully responsive (desktop + mobile)  

---

## ⚙️ Tech Stack  

- HTML5  
- CSS3 (Flexbox, Grid, Animations)  
- JavaScript (DOM manipulation, timers)  
- React (component-based version included)
